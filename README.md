# 🌿 Bagicha - Smart Plant Selection Web App

💡 A modern frontend project showcasing real-world features like filtering, theme toggle, and client-side data persistence.

---

## 📌 Overview

Bagicha is a responsive multi-page web application designed to help users explore, select, and manage plants for their home environment.
The project focuses on enhancing user experience through dynamic UI, real-time interactions, and personalized features.

---

## 🚀 Features

### 🌱 Core Features

* Browse and explore different types of plants
* Clean and responsive UI for all devices
* Multi-page navigation

### 🔍 Advanced Functionality

* Real-time search for quick plant discovery
* Category-based filtering system
* Dark/Light mode toggle for better accessibility

### 🪴 Personalization

* "My Garden" feature to save selected plants
* Data stored using LocalStorage (no backend required)

### ✅ Form & UX Improvements

* Input validation for better data handling
* Smooth UI interactions and transitions

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (ES6+)
* LocalStorage

## ⚙️ How It Works

* Users can browse plants and filter them based on categories
* Search functionality dynamically updates results
* Selected plants are stored in LocalStorage
* Theme toggle allows switching between dark and light modes

3. Run `index.html` in your browser
